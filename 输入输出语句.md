JavaScript 不提供任何内建的打印或显示函数。

## document.write()

使用`document.write()`写入 HTML 输出

- 示例

```javascript
document.write("光荣在于平淡，艰巨在于漫长。");
```

## window\.alert()

使用`window.alert()`写入警告框

```javascript
window.alert("光荣在于平淡，艰巨在于漫长。");
```

- 效果

## innerHTML

使用`innerHTML`写入 HTML 元素

- 示例

```javascript
document.getElementById("元素ID").innerHTML = 输出内容;
```

- 实例

```javascript
<body>
  <p>光荣在于平淡，艰巨在于漫长。</p>
  <p id="demo">光荣在于平淡，艰巨在于漫长。</p>
  <script>
      document.getElementById("demo").innerHTML = "成功在于坚持";
  </script>
</body>
```

- 效果：

![](https://raw.githubusercontent.com/codesports/JavaScript/refs/heads/main/public/003.png)
